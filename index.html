O que é o atributo ROWSPAN?

O atributo ROWSPAN é usado em tabelas HTML para especificar o número de linhas que uma célula deve ocupar.

O que é o atributo COLSPAN?

O atributo COLSPAN permite que uma célula de tabela se estenda por várias colunas.
